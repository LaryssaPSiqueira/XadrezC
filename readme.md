# ![XadrezC](https://img.shields.io/badge/XadrezC-Jogo%20de%20Xadrez%20em%20C-6A0DAD?style=for-the-badge&logo=github&logoColor=white)

Este repositório reúne o projeto **XadrezC**, desenvolvido em linguagem C para simular os movimentos das peças de xadrez por meio de desafios progressivos de programação.

O projeto é dividido em três níveis de complexidade: **Novato, Aventureiro e Mestre**.

---

## ![Conteúdo](https://img.shields.io/badge/O%20que%20tem%20aqui-N%C3%ADveis%20do%20Projeto-000000?style=for-the-badge)

### ♟️ Nível Novato  
Simula os movimentos da **Torre, Bispo e Rainha** utilizando estruturas de repetição básicas (`for`, `while` e `do-while`).  
O foco é praticar lógica de programação, laços de repetição e saída de dados no console.

---

### ♟️ Nível Aventureiro  
Adiciona a movimentação do **Cavalo**, que se desloca em formato de **"L"**, utilizando **loops aninhados**.  
O objetivo é aprofundar o uso de estruturas de repetição e controle de fluxo do programa.

---

### ♟️ Nível Mestre  
Aprimora o projeto com o uso de **recursividade** e **loops mais complexos** para simular os movimentos das peças.  
Este nível consolida conceitos avançados da linguagem C, como organização do código, funções recursivas e maior controle lógico.
